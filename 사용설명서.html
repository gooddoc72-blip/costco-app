2. 처음 시작 — 필수 설정

프로그램을 처음 사용할 때 ⚙️ 설정 메뉴에서 아래 항목들을 먼저 입력해야 합니다.

2-1. 엑셀 비밀번호 설정

네이버 스마트스토어에서 주문 엑셀을 다운받으면 비밀번호가 걸려 있습니다. 이 비밀번호를 한 번만 저장해두면 이후 자동으로 해제됩니다.

STEP 1
네이버 스마트스토어 센터에서 비밀번호 확인
스마트스토어 센터 로그인 → 주문 관리 → 발주확인/발송관리 → 엑셀 다운로드 시 표시되는 비밀번호 확인
STEP 2
프로그램 설정에 입력
⚙️ 설정 메뉴 → 🔓 엑셀 비밀번호 → 비밀번호 입력 → 비밀번호 저장 클릭

2-2. 네이버 커머스 API 설정

주문 자동 조회와 송장 자동 등록을 사용하려면 네이버 API 키가 필요합니다.

STEP 1
네이버 커머스 API 키 발급
  1. 네이버 스마트스토어 센터 로그인
  2. 우측 상단 계정 이름 클릭 → 커머스API센터
  3. 애플리케이션 등록 → 앱 이름 입력 (예: 코스트코관리)
  4. 권한 체크: 주문 상품 발송
  5. 등록 완료 후 애플리케이션 ID와 시크릿 복사
STEP 2
프로그램 설정에 입력
⚙️ 설정 → 🔗 네이버 커머스 API → 애플리케이션 ID / 시크릿 입력 → API 키 저장
✅ API 연결 성공! 메시지가 뜨면 완료

2-3. 카카오톡 알림 설정 ⭐

장보기 목록을 본인 카카오톡으로 전송하는 기능입니다. 아래 순서를 그대로 따라해 주세요.

⚠️ 준비물: 카카오 계정으로 developers.kakao.com 접속 가능한 상태여야 합니다.
STEP 1
카카오 개발자 사이트에서 앱 만들기
  1. developers.kakao.com 접속 → 카카오 계정으로 로그인
  2. 상단 메뉴 내 애플리케이션 클릭
  3. 애플리케이션 추가하기 클릭
  4. 앱 이름: 코스트코핫딜 (아무 이름이나 가능), 회사명 입력 → 저장
STEP 2
REST API 키 복사
  1. 방금 만든 앱 클릭 → 앱 키 섹션에서
  2. REST API 키 옆의 값을 복사 (예: a1b2c3d4e5f6g7h8...)
💡 REST API 키는 32자리 영문+숫자 조합입니다.
STEP 3
플랫폼 및 카카오 로그인 설정
  1. 앱 왼쪽 메뉴 → 플랫폼 → Web 플랫폼 등록
  2. 사이트 도메인: http://localhost 입력 → 저장
  3. 왼쪽 메뉴 → 카카오 로그인 → 활성화 스위치 ON
  4. Redirect URI 등록 → http://localhost → 저장
STEP 4
동의항목 설정
  1. 왼쪽 메뉴 → 카카오 로그인 → 동의항목
  2. 카카오톡 메시지 전송 항목 찾기 → 설정 클릭 → 선택 동의 선택 → 저장
STEP 5
프로그램 설정에 REST API 키 입력
  1. 프로그램의 ⚙️ 설정 메뉴 → 📱 카카오톡 알림
  2. REST API 키 입력창에 STEP 2에서 복사한 키 붙여넣기
  3. REST API 키 저장 클릭
STEP 6
카카오 로그인 후 인가 코드 받기
  1. STEP 5 완료 후 화면에 "1단계: 여기를 클릭하여 카카오 로그인" 링크가 나타납니다
  2. 해당 링크 클릭 → 브라우저에서 카카오 로그인 → 동의하고 계속하기
  3. 동의 후 브라우저 주소창이 바뀝니다:
http://localhost?code=abc123xyz456def789...

주소창에서 code= 뒤의 값을 전체 복사합니다.

🚨 주의: 인가 코드는 10분 내에 사용해야 합니다. 시간이 지나면 다시 STEP 6부터 반복해야 합니다.
STEP 7
인가 코드 입력 후 토큰 발급
  1. 프로그램 설정 화면의 "2단계: 인가 코드 붙여넣기" 칸에 복사한 코드 붙여넣기
  2. 🔑 토큰 발급받기 버튼 클릭
  3. ✅ 토큰 발급 성공! 메시지 확인
💡 토큰은 자동으로 저장되므로 한 번만 하면 됩니다. 만료되면 자동 갱신됩니다.
STEP 8
카카오톡 테스트 전송
  1. 🔔 카카오톡 테스트 전송 버튼 클릭
  2. 본인 카카오톡에 "✅ 코스트코핫딜 알림 테스트 성공!" 메시지 도착 확인

3. 주문 업로드 사용법

네이버 스마트스토어에서 주문 엑셀을 다운받아 업로드합니다.

네이버 스마트스토어에서 엑셀 다운로드

  1. 스마트스토어 센터 → 주문 관리 → 발주확인/발송관리
  2. 조회 기간 설정 → 검색
  3. 엑셀 다운로드 클릭 → 파일 저장

프로그램에 업로드

  1. 📋 주문 업로드 메뉴 클릭
  2. 파일 선택 또는 파일을 드래그하여 업로드
  3. 자동으로 비밀번호 해제 후 주문 목록이 표시됩니다
  4. 목록에서 각 주문의 구입가격을 확인하고 필요 시 수정
💡 노란색 행은 묶음 상품 (코스트코 구매수량 = 주문수량 × 묶음수량)입니다.

장보기 목록 카카오톡 전송

주문 목록 하단의 📱 장보기 목록 휴대폰 전송 버튼을 클릭하면 코스트코에서 구입해야 할 목록을 카카오톡으로 전송합니다.

⚠️ 카카오톡 설정이 완료된 경우에만 전송됩니다. 설정이 안 됐다면 2-3 항목을 먼저 완료하세요.

4. 수익 계산 및 저장

주문 업로드 후 💰 수익 계산 탭에서 각 주문의 수익을 계산하고 저장합니다.

수익 계산 공식

수익 = (정산예정금액 + 고객택배비) - (구입가격 + 택배비 + 박스비)

사용 순서

  1. 주문 업로드 완료 후 💰 수익 계산 탭 클릭
  2. 각 상품의 구입가격이 올바른지 확인 (잘못된 경우 직접 수정 가능)
  3. 수익 확인 후 💾 오늘 날짜로 저장 클릭
  4. 저장 완료 후 📊 대시보드에서 월별 통계 확인 가능

5. 카카오톡 장보기 목록 전송

주문 업로드 후 코스트코에서 구입해야 할 상품 목록을 카카오톡으로 전송하는 기능입니다.

전송되는 내용

📋 코스트코 장보기 목록 ────────────────────── ✅ 커클랜드 가염버터 453g x4 4개 ✅ 코스트코 그릭요거트 907g 6개 ✅ 라페크미에르 요거트 140g×2 5개 ... ────────────────────── 🛒 예상 구매금액: 943,100원 📦 단가 미등록 상품: 2종

사용 방법

  1. 📋 주문 업로드 탭에서 주문 엑셀 업로드
  2. 목록 하단 📱 장보기 목록 휴대폰 전송 버튼 클릭
  3. 내 카카오톡으로 목록 수신 확인
💡 자동 전송도 가능합니다!
🤖 자동화 메뉴에서 매일 정해진 시간에 자동으로 전송되도록 설정할 수 있습니다.

6. 네이버 상품 등록 (🛍 네이버 등록)

코스트코 제품을 네이버 스마트스토어에 등록합니다. 4단계 과정으로 진행됩니다.

등록 전 확인사항

  • 네이버 커머스 API 설정 완료 (2-2 항목)
  • 제품 DB에 상품이 등록되어 있어야 함

4단계 등록 과정

STEP 1
코스트코 카테고리 선택
등록할 상품의 코스트코 카테고리 버튼을 클릭합니다. (예: 냉장/냉동, 식품, 생활용품 등)
STEP 2
네이버 카테고리 검색 및 선택
  1. 검색창에 카테고리 키워드 입력 (예: "버터", "요거트")
  2. 검색 결과에서 적합한 네이버 카테고리 선택
  3. ✅ 이 카테고리로 설정 클릭
STEP 3
키워드 입력 및 AI 추천
  1. 상품 검색 키워드 입력 (예: "버터", "가염")
  2. AI 추천 버튼 클릭
  3. AI가 카테고리와 잘 맞는 상품을 점수별로 표시합니다:
    🟢 매우 적합   🟡 적합   ⚪ 참고용
STEP 4
상품 선택 후 장바구니 추가 → 일괄 등록
  1. 등록할 상품에 체크박스 선택
  2. ➕ 장바구니에 추가 클릭 (최대 30개 누적 가능)
  3. 다른 카테고리 상품도 같은 방법으로 장바구니에 추가
  4. 준비 완료 후 상단의 🚀 일괄 등록 클릭
💡 장바구니 방식이라 여러 카테고리의 상품을 모아서 한 번에 등록할 수 있습니다.

7. 자동화 설정 (🤖 자동화)

매일 정해진 시간에 자동으로 주문 조회 및 카카오톡 전송이 실행됩니다.

장보기 목록 자동 전송 설정

  1. 🤖 자동화 메뉴 클릭
  2. Task 1 — 장보기 목록 카카오 발송 섹션
  3. 실행 시간 설정 (예: 매일 오전 9시)
  4. 자동화 활성화 토글 ON
⚠️ 자동화는 프로그램이 실행 중인 상태에서만 작동합니다. PC를 끄면 자동화가 멈춥니다.

8. 자주 묻는 질문 (FAQ)

❓ 카카오톡 전송이 안돼요

  • ⚙️ 설정 → 카카오톡 알림 → ✅ 액세스 토큰 설정됨 표시 확인
  • 토큰이 없으면 2-3 항목의 설정 과정을 다시 진행
  • 🔔 카카오톡 테스트 전송 버튼으로 먼저 테스트

❓ 인가 코드가 만료됐다고 나와요

  • 인가 코드는 발급 후 10분이 지나면 사용 불가
  • 설정의 "1단계: 카카오 로그인" 링크를 다시 클릭하여 새 코드 발급

❓ 엑셀 파일이 안 열려요

  • ⚙️ 설정 → 엑셀 비밀번호 확인
  • 스마트스토어 센터에서 현재 비밀번호를 다시 확인 후 재입력

❓ 네이버 API 연결 실패가 뜨면?

  • 커머스API센터에서 애플리케이션 ID / 시크릿을 다시 복사하여 입력
  • API 권한이 승인되지 않았을 수 있음 → 커머스API센터에서 승인 상태 확인

❓ 상품이 '분류불가'로 나와요

  • 📦 제품 DB에서 해당 상품의 카테고리를 직접 지정
  • 또는 🛍 네이버 등록에서 네이버 카테고리 선택 후 키워드 검색으로 찾기

❓ 카카오 토큰은 얼마나 유지되나요?

  • 액세스 토큰: 12시간
  • 리프레시 토큰: 60일
  • 프로그램이 자동으로 갱신하므로 보통 다시 설정할 필요 없음
  • 60일이 지나면 2-3 항목의 설정을 다시 진행
💡 문의: 사용 중 문제가 발생하면 화면 캡처와 함께 관리자에게 문의해주세요.
