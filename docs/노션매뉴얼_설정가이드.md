# 1. 네이버 커머스 API (주문 조회 · 자동 발송처리)

주문 자동 조회, 발송처리 자동화에 필요합니다. **스마트스토어 판매자 계정**이 있어야 합니다.

> ⚠️ 핵심은 **애플리케이션 ID / 시크릿** 입니다. (판매자 ID는 참고용)

### STEP 1. 네이버 커머스 API 센터 접속
브라우저에 아래 주소를 입력합니다.
```
https://apicenter.commerce.naver.com
```
> 🖼️ [이미지: 커머스 API 센터 첫 화면]

### STEP 2. 본인 스마트스토어 계정으로 로그인
**반드시 본인의** 네이버 스마트스토어 판매자 계정으로 로그인합니다. (이 계정의 상품이 수집됩니다)
> 🖼️ [이미지: 로그인 화면]

### STEP 3. 애플리케이션 등록
상단 메뉴 **애플리케이션 관리 → 애플리케이션 등록** 클릭

| 입력 항목 | 입력값 |
|---|---|
| 애플리케이션 이름 | 코코비즈 (자유롭게) |
| 사용 API | ✅ **스마트스토어 API** 체크 |
| Callback URL | `http://localhost` |

> 🖼️ [이미지: 애플리케이션 등록 화면 — 스마트스토어 API 체크된 모습]

### STEP 4. Client ID / Client Secret 복사
등록 완료 후 **애플리케이션 상세** 페이지에서 두 값을 복사합니다.
- **Client ID** → ⚙️ 설정 탭 > 네이버 커머스 API > **애플리케이션 ID** 에 붙여넣기
- **Client Secret** → **애플리케이션 시크릿** 에 붙여넣기

> 🖼️ [이미지: 애플리케이션 상세 — Client ID / Secret 위치 표시]

### STEP 5. 판매자 채널 ID 확인 (참고용 · 선택)
스마트스토어 센터(smartstore.naver.com) → **외부서비스 연동 → API 관리** 에서 확인
- **채널 API 연동용 판매자ID** 복사 → ⚙️ 설정 탭 > **API 연동용 판매자ID** 에 붙여넣기

> ⚠️ 판매자 ID는 **참고용**입니다. 상품·주문 수집은 위 애플리케이션 ID/시크릿으로만 이뤄집니다.

### STEP 6. ⚙️ 설정 탭에 입력 후 저장
저장 버튼을 누르면 자동으로 **API 연결 테스트**가 진행됩니다.
- ✅ "API 연결 성공!" 메시지가 뜨면 완료
- ❌ 실패 시: Client Secret 재확인 / API 권한에 **스마트스토어 API** 체크 확인

> 🖼️ [이미지: 설정 탭 — 커머스 API 입력 & 연결 성공 메시지]

# 2. 카카오톡 알림 (장보기 목록 · 자동화 알림)

자동화 작업 완료 후 카카오톡 **나에게 보내기**로 알림을 받습니다.

### STEP 1. 카카오 개발자 센터 접속
```
https://developers.kakao.com
```
> 🖼️ [이미지: 카카오 개발자 센터 첫 화면]

### STEP 2. 로그인 & 앱 만들기
카카오 계정 로그인 → 상단 **내 애플리케이션 → 애플리케이션 추가하기**

| 입력 항목 | 입력값 |
|---|---|
| 앱 이름 | 코코비즈 (자유롭게) |
| 사업자명 | 본인 이름 또는 상호 |

> 🖼️ [이미지: 애플리케이션 추가 팝업]

### STEP 3. REST API 키 복사
앱 생성 후 **앱 키** 섹션에서 **REST API 키** 복사
- ⚙️ 설정 탭 > 카카오톡 알림 > **REST API 키** 에 붙여넣기 → **REST API 키 저장** 클릭

> 🖼️ [이미지: 앱 키 화면 — REST API 키 위치]

### STEP 4. 플랫폼 등록
왼쪽 메뉴 **앱 설정 → 플랫폼 → Web 플랫폼 등록**
- **사이트 도메인**: `http://localhost` 입력 후 저장

> ⚠️ 이 단계를 빠뜨리면 토큰 발급이 실패합니다.
> 🖼️ [이미지: Web 플랫폼 등록 — localhost 입력]

### STEP 5. 카카오 로그인 활성화
왼쪽 메뉴 **제품 설정 → 카카오 로그인**
1. **활성화 설정** → **ON**
2. **Redirect URI** → `http://localhost` 추가
3. **동의항목** 탭 → **카카오톡 메시지 전송** → **필수 동의** 설정

> 🖼️ [이미지: 카카오 로그인 활성화 ON + Redirect URI]
> 🖼️ [이미지: 동의항목 — 카카오톡 메시지 전송 필수 동의]

### STEP 6. ⚙️ 설정 탭에서 인가 코드 발급 & 토큰 연결
1. ⚙️ 설정 탭 > 카카오톡 알림 > **[여기를 클릭하여 카카오 로그인]** 클릭
2. 카카오 로그인 후 **동의하고 계속** 클릭
3. 브라우저 주소창에서 `http://localhost?code=` **뒤의 코드** 복사
   - 예: `http://localhost?code=` **abc123xyz**
4. 코드를 **인가 코드 붙여넣기** 칸에 붙여넣기 → **🔑 토큰 발급받기** 클릭

> ✅ "액세스 토큰 설정됨" → **🔔 카카오톡 테스트 전송**으로 최종 확인
> 💡 토큰은 6시간마다 자동 갱신됩니다. 한 번 설정하면 끝.
> ⚠️ 인가 코드는 **1회용**입니다. 오류 시 1단계 링크를 다시 클릭해 새 코드를 받으세요.

> 🖼️ [이미지: 설정 탭 — 인가 코드 입력 & 토큰 발급 성공]

# 3. 네이버 Open API (키워드 순위 체크)

키워드 순위 체크용 API입니다. **네이버 커머스 API와 별개**로 발급합니다.

### STEP 1. 네이버 개발자 센터 접속
```
https://developers.naver.com
```

### STEP 2. 로그인 & 애플리케이션 등록
네이버 계정 로그인 → 상단 **Application → 애플리케이션 등록**

### STEP 3. 사용 API 선택

| 체크 항목 | 설명 |
|---|---|
| ✅ **검색 > 쇼핑** | 쇼핑 검색 결과 조회 (순위 체크) |

- **Callback URL**: 공란 또는 `http://localhost`
- **서비스 환경**: `PC` 체크

> 🖼️ [이미지: 애플리케이션 등록 — 검색>쇼핑 체크, PC 환경]

### STEP 4. Client ID / Client Secret 복사
**내 애플리케이션**에서 방금 만든 앱 클릭
- **Client ID** → ⚙️ 설정 탭 > 네이버 Open API > **Client ID**
- **Client Secret** → **Client Secret** 에 붙여넣기 → **Open API 저장** 클릭

> ✅ 저장 후 📈 순위 체크 탭 → **🧪 API 키 테스트**로 확인
> 💡 무료 사용량: 하루 25,000건. 키워드 수십 개를 매일 체크해도 충분합니다.

> 🖼️ [이미지: 설정 탭 — Open API 입력]

# 4. 쿠팡 Wing Open API (쿠팡 주문 조회) — 선택

쿠팡 Wing 판매자 계정이 있어야 합니다.

### STEP 1. 쿠팡 Wing 개발자 센터 접속
```
https://wing.coupang.com
```
로그인 후 상단 **개발자 센터 → Open API**

### STEP 2. Access Key / Secret Key 발급
**API Key 발급** 클릭 → 새 키 생성
- **Access Key** → ⚙️ 설정 탭 > 쿠팡 Wing > **Access Key**
- **Secret Key** → **Secret Key** 에 붙여넣기

> ⚠️ Secret Key는 발급 즉시 **한 번만** 표시됩니다. 바로 복사하세요.

### STEP 3. Vendor ID 확인
Wing 로그인 후 브라우저 주소창 URL에서 확인
- 예: `https://wing.coupang.com/vendor/` **A00012345** `/dashboard`
- `A`로 시작하는 값을 복사 → **Vendor ID** 에 붙여넣기

> 🖼️ [이미지: Wing 주소창 — Vendor ID 위치]

### STEP 4. ⚙️ 설정 탭에 입력 후 저장
저장 시 자동으로 연결 테스트가 진행됩니다.
> ✅ 성공 시 📋 일일 주문 수집 탭에서 **🛒 쿠팡 주문 조회** 사용 가능

# 5. 기타 설정

### 🔓 엑셀 비밀번호
1. 네이버 스마트스토어 센터에서 주문 엑셀 다운로드 (발주 확인/발송 관리)
2. 엑셀 파일을 열면 나오는 **비밀번호** 확인
3. ⚙️ 설정 탭 > 엑셀 비밀번호에 입력 후 저장 → 이후 업로드 시 자동 해제

> 💡 비밀번호는 네이버 판매자 계정마다 다를 수 있습니다.

### 📦 고정 비용
- **택배비**: 실제 납부 택배비 (기본 1,800원)
- **박스비**: 포장 박스 단가 (기본 300원)
- 수익 계산 시 자동 차감됩니다.

### 📱 텔레그램 알림 (선택)
카카오톡 대신 텔레그램으로도 알림 가능.
1. 텔레그램에서 **@BotFather** → `/newbot` → **봇 토큰** 복사
2. **@userinfobot** 에 메시지 발송 → **Chat ID** 확인
3. ⚙️ 설정 탭 > 텔레그램에 입력 후 저장

> 매일의 작업 흐름: **주문 수집 → (다음 날) 영수증·수익 계산 → 송장 발송 → 정산 매칭**

### 🏠 홈
이번 달 주문금액·수익 요약(KPI)과 빠른 이동 버튼이 있습니다.

1. **네이버 주문 상태/기간** 선택 → **네이버 주문 조회**
2. (쿠팡 사용 시) **쿠팡 주문 조회**
3. 목록 검토 후 **💾 저장** — 저장해야 수익 계산에 반영됩니다.
> ⚠️ 저장하지 않으면 데이터가 사라집니다.

> 🖼️ [이미지: 송장번호 — 파일 업로드 & 발송처리]

### 💰 수익 계산
1. **주문 날짜 선택** → 그날 저장된 주문 로드
2. **영수증 등록**(코스트코 영수증 PDF) → 매입가 자동 매칭
3. 행별 단가/발송비/박스비 확인 → **정산 저장**
> 🖼️ [이미지: 수익 계산 표]

### 💳 정산 매칭
네이버 정산 CSV(스마트스토어 → 정산관리 → 빠른정산 건별 다운로드) 업로드 → 발송건과 매칭.
> 🖼️ [이미지: 정산 매칭]

### 📊 대시보드
월별 달력으로 일별 주문/수익/정산을 확인합니다.

### 📦 제품 DB / 🛍 네이버 등록 / 📈 순위 체크
- **제품 DB**: 등록 상품·매입가 관리
- **네이버 등록**: 본인 스토어 상품 수집
- **순위 체크**: 키워드 순위 추적 (Open API 필요)

### 🤖 자동화
주문 수집·장보기 발송·순위 체크·상품 수집을 **정해진 시간에 자동 실행**하도록 등록합니다.

# 7. 자주 묻는 질문 (FAQ)

**Q. 카카오톡 토큰 발급 시 'redirect_uri mismatch' 오류가 납니다.**
> 카카오 로그인 → Redirect URI에 `http://localhost`가 **정확히**(끝에 `/` 없이) 등록됐는지 확인하세요.

**Q. 인가 코드를 넣었는데 '이미 사용된 코드'라고 나옵니다.**
> 인가 코드는 1회용입니다. 설정 탭에서 카카오 로그인 링크를 다시 클릭해 새 코드를 받으세요.

**Q. 네이버 커머스 API 저장 후 '401 Unauthorized' 오류.**
> Client Secret이 틀렸거나, 스마트스토어 API 권한이 미체크된 경우입니다.

**Q. 순위 체크 시 '검색 결과 없음'.**
> 키워드를 더 짧고 일반적인 검색어로 바꿔 보세요.

**Q. 엑셀 업로드 시 '비밀번호가 틀렸습니다'.**
> 스마트스토어 로그인 비밀번호와 **엑셀 비밀번호는 다릅니다.** 설정 탭의 엑셀 비밀번호를 재확인하세요.

**Q. CJ대한통운 자동 접수가 실패합니다.**
> 설정 탭 > 택배사 설정에서 CJ ID/PW/고객번호 입력 확인. 고객번호는 CJ 기업 고객센터(1588-1255) 문의.
