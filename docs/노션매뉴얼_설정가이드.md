# 📘 코코비즈(코스트코 자동화) 사용 매뉴얼

> 컴퓨터·API가 처음인 분도 **그대로 따라만 하면** 끝나도록 만들었습니다.
> 어려운 말은 바로 옆에 풀어서 설명해 두었어요. 천천히 순서대로 해보세요.
> 각 `🖼️ [이미지]` 자리에 해당 화면 스크린샷을 넣어 주세요.

---

## 🧭 시작하기 전에 — 딱 3가지만 알고 가세요

**1) 이 프로그램은 무엇을 하나요?**
네이버·쿠팡 **주문을 자동으로 모으고**, 코스트코 영수증과 맞춰 **수익을 계산**하고, **송장(택배번호)을 일괄 등록**하고, 카카오톡으로 **장보기 목록을 보내주는** 프로그램입니다.

**2) 'API 키'가 뭔가요?** 🔑
프로그램이 내 네이버·쿠팡·카카오 계정에 **대신 접속할 수 있게 해주는 '출입증'** 입니다.
- **ID(아이디)** = 누구인지 알려주는 이름표
- **Secret(시크릿)/Key(키)** = 비밀번호 같은 것 → **남에게 절대 보여주지 마세요**

> 💡 **핵심**: 아래에서 하는 일은 전부 "각 사이트에서 출입증(키)을 발급받아 → 프로그램 설정에 붙여넣기" 입니다. 그게 전부예요.

**3) 가장 중요한 팁**
키 값은 **손으로 타이핑하지 말고 꼭 '복사(Ctrl+C) → 붙여넣기(Ctrl+V)'** 하세요. 한 글자만 틀려도 연결이 안 됩니다.

# 1️⃣ 네이버 커머스 API — 주문 자동으로 가져오기

> 📌 **왜 필요한가요?** 이 키가 있어야 프로그램이 내 스마트스토어 주문을 자동으로 불러오고, 발송처리를 대신 해줍니다.
> 🎒 **준비물**: 네이버 스마트스토어 판매자 계정(아이디/비번)
> ⏱ **약 10분**

### STEP 1. 사이트 접속
인터넷 주소창에 아래를 복사해서 붙여넣고 Enter
```
https://apicenter.commerce.naver.com
```
> 🖼️ [이미지: 커머스 API 센터 첫 화면]

### STEP 2. 로그인
**내 스마트스토어 판매자 계정**으로 로그인합니다. (다른 계정으로 하면 안 돼요 — 이 계정의 상품이 들어옵니다)
> 🖼️ [이미지: 로그인 화면]

### STEP 3. 애플리케이션(앱) 등록하기
> 💬 '애플리케이션 등록' = "내 프로그램이 쓸 출입증을 새로 만든다"는 뜻이에요.

> ✅ **개발 업체(솔루션사)가 아니어도 됩니다.** 일반 판매자가 본인 스마트스토어 계정으로 **'내 스토어 애플리케이션'을 직접** 등록해 바로 쓸 수 있어요. (전부 **무료**)
> ⚠️ 등록 중 유형을 묻거든 **'내 스토어용 / 주문 판매자 API'** 쪽을 고르세요. **'커머스솔루션'은 외부 개발 업체 전용**이니 선택하지 마세요.

화면 위쪽 메뉴에서 **애플리케이션 관리** → **애플리케이션 등록** 을 누르고, 아래처럼 입력하세요.

| 입력칸 | 무엇을 넣나요 |
|---|---|
| 애플리케이션 이름 | `코코비즈` (아무 이름이나 OK) |
| 사용 API | ✅ **스마트스토어 API** 에 체크 (이거 꼭!) |
| Callback URL | `http://localhost` 그대로 입력 |

> ⚠️ **자주 하는 실수**: '스마트스토어 API' 체크를 빠뜨리면 나중에 연결이 안 됩니다. 꼭 체크하세요.
> 🖼️ [이미지: 앱 등록 화면 — 스마트스토어 API에 체크된 모습 (빨간 네모로 표시)]

### STEP 4. 출입증(키) 2개 복사하기
등록하면 **애플리케이션 상세** 화면에 두 값이 보입니다.

| 보이는 값 | 어디에 붙여넣나요 |
|---|---|
| **Client ID** (애플리케이션 ID) | 설정 탭 > **애플리케이션 ID** |
| **Client Secret** (애플리케이션 시크릿) | 설정 탭 > **애플리케이션 시크릿** |

> 💡 Secret은 '보기' 버튼을 눌러야 보이는 경우가 있어요. 복사해서 메모장에 잠깐 붙여두면 편합니다.
> 🖼️ [이미지: Client ID / Secret 위치 (빨간 네모로 표시)]

### STEP 5. (선택) 판매자 ID — 안 해도 됩니다
참고용이라 **건너뛰어도 정상 작동**합니다. 필요하면: 스마트스토어센터 → 외부서비스 연동 → API 관리에서 '판매자ID' 복사.

### STEP 6. 프로그램에 붙여넣고 저장
프로그램 왼쪽 **⚙️ 설정** 탭 → 네이버 커머스 API 칸에 위 두 값을 붙여넣고 **저장**.

> ✅ **이렇게 되면 성공**: "API 연결 성공!" 초록색 메시지가 뜸
> ❌ **실패하면**: ① Secret을 다시 복사 ② STEP 3에서 '스마트스토어 API' 체크했는지 확인
> 🖼️ [이미지: 설정 탭 — 입력 후 '연결 성공' 메시지]

# 2️⃣ 카카오톡 — 알림 받기

> 📌 **왜 필요한가요?** 장보기 목록·작업 완료 알림을 **내 카카오톡(나에게 보내기)** 으로 받기 위해서예요.
> 🎒 **준비물**: 카카오 계정
> ⏱ **약 15분** (조금 길지만 한 번만 하면 됩니다)

### STEP 1. 사이트 접속
```
https://developers.kakao.com
```
> 🖼️ [이미지: 카카오 개발자 센터 첫 화면]

### STEP 2. 로그인 → 앱 만들기
카카오 계정으로 로그인 → 위쪽 **내 애플리케이션** → **애플리케이션 추가하기**

| 입력칸 | 무엇을 넣나요 |
|---|---|
| 앱 이름 | `코코비즈` (아무거나) |
| 사업자명 | 본인 이름 또는 가게 이름 |

> 🖼️ [이미지: 앱 추가 팝업]

### STEP 3. REST API 키 복사
앱을 만들면 **앱 키** 화면이 나옵니다. 그중 **REST API 키** 를 복사하세요.
→ 프로그램 ⚙️ 설정 탭 > 카카오톡 > **REST API 키** 에 붙여넣고 **저장**.

> 💬 'REST API 키'가 여러 키 중 하나예요. 다른 키(JavaScript 키 등) 말고 **REST API 키** 를 복사해야 합니다.
> 🖼️ [이미지: 앱 키 화면 — REST API 키 (빨간 네모)]

### STEP 4. 플랫폼 등록 (이거 빠뜨리면 안 돼요!)
왼쪽 메뉴 **앱 설정 → 플랫폼** → **Web 플랫폼 등록** 버튼
- **사이트 도메인** 칸에 `http://localhost` 입력 후 저장

> ⚠️ 이 단계를 빠뜨리면 다음 단계(토큰 발급)가 **무조건 실패**합니다. 꼭 하세요.
> 🖼️ [이미지: Web 플랫폼 — localhost 입력]

### STEP 5. 카카오 로그인 켜기 + 메시지 권한 주기
왼쪽 메뉴 **제품 설정 → 카카오 로그인**
1. 맨 위 **활성화 설정** 을 **ON**(켜기) 으로
2. **Redirect URI** 에 `http://localhost` 추가
3. **동의항목** 탭 → **카카오톡 메시지 전송** → **필수 동의** 로 설정

> 💬 이건 "이 프로그램이 내 카톡으로 메시지 보내는 걸 허락한다"는 설정이에요.
> 🖼️ [이미지: 로그인 활성화 ON + Redirect URI]
> 🖼️ [이미지: 동의항목 — 카카오톡 메시지 전송 '필수 동의']

### STEP 6. 마지막 연결 (프로그램에서 진행)
프로그램 ⚙️ 설정 탭 > 카카오톡에서:
1. **[여기를 클릭하여 카카오 로그인]** 누르기
2. 카카오 로그인 후 **동의하고 계속** 누르기
3. 그러면 빈 화면이 뜨는데, **주소창**을 보세요:
   `http://localhost/?code=` **여기에 붙은 글자(코드)** 를 복사
   (예: `...?code=` **abc123xyz** ← 이 부분)
4. 복사한 코드를 **인가 코드** 칸에 붙여넣고 **🔑 토큰 발급받기** 클릭

> ✅ **성공**: "액세스 토큰 설정됨" → **🔔 카카오톡 테스트 전송** 눌러서 내 카톡에 메시지 오면 완료!
> ⚠️ **코드는 1번만 쓸 수 있어요.** 오류 나면 1번부터 다시(새 코드 발급) 하면 됩니다.
> 💡 한 번 연결하면 6시간마다 **자동 갱신**돼서 다시 설정할 필요 없습니다.
> 🖼️ [이미지: 주소창의 code= 부분 (빨간 네모) + 설정 탭 토큰 발급]

# 3️⃣ 네이버 Open API — 키워드 순위 체크

> 📌 **왜 필요한가요?** 내 상품이 네이버 쇼핑 검색에서 **몇 위인지** 매일 확인하려면 필요합니다. (안 쓰면 건너뛰어도 됩니다)
> 🎒 **준비물**: 네이버 계정 (스마트스토어 아니어도 됨)
> ⏱ **약 7분**

> ⚠️ 이건 1번(커머스 API)과 **완전히 다른 키**입니다. 헷갈리지 마세요. (사이트도 다릅니다)

### STEP 1. 사이트 접속
```
https://developers.naver.com
```

### STEP 2. 로그인 → 애플리케이션 등록
네이버 로그인 → 위쪽 **Application** → **애플리케이션 등록**

### STEP 3. 사용할 기능 선택
- **사용 API**: **검색** 선택 (그 안에서 '쇼핑'이 쓰입니다)
- **서비스 환경**: **PC** 에 체크
- **Callback URL**: 비워두거나 `http://localhost`

> 🖼️ [이미지: 앱 등록 — 검색 선택, PC 체크]

### STEP 4. 키 2개 복사 → 프로그램에 입력
**내 애플리케이션**에서 방금 만든 앱을 누르면 키가 보입니다.

| 보이는 값 | 어디에 붙여넣나요 |
|---|---|
| **Client ID** | 설정 탭 > 네이버 Open API > **Client ID** |
| **Client Secret** | 설정 탭 > **Client Secret** |

붙여넣고 **Open API 저장** 클릭.

> ✅ 저장 후 📈 **순위 체크** 탭 → **🧪 API 키 테스트** 가 성공이면 완료
> 💡 무료로 하루 25,000번까지 조회 가능 — 매일 써도 차고 넘칩니다.

# 4️⃣ 쿠팡 Wing API — 쿠팡도 파는 경우만 (선택)

> 📌 **왜 필요한가요?** 쿠팡 주문도 프로그램으로 같이 관리하려면 필요합니다. 쿠팡 안 팔면 건너뛰세요.
> 🎒 **준비물**: 쿠팡 Wing 판매자 계정
> ⏱ **약 7분**

### STEP 1. 사이트 접속 → 개발자 센터
```
https://wing.coupang.com
```
로그인 후 위쪽 **개발자 센터 → Open API**

### STEP 2. 키 발급
**API Key 발급** 버튼 → 새 키 생성
- **Access Key** → 설정 탭 > 쿠팡 Wing > **Access Key**
- **Secret Key** → 설정 탭 > **Secret Key**

> ⚠️ **Secret Key는 발급할 때 딱 한 번만 보입니다.** 화면 닫기 전에 꼭 복사하세요. (놓치면 새로 발급해야 해요)

### STEP 3. Vendor ID(판매자 번호) 찾기
쿠팡 Wing에 로그인한 상태에서 **주소창 URL**을 보세요.
- 예: `https://wing.coupang.com/vendor/` **A00012345** `/dashboard`
- `A`로 시작하는 가운데 부분이 Vendor ID → 복사해서 설정 탭 > **Vendor ID** 에 붙여넣기

> 🖼️ [이미지: 주소창 — Vendor ID 위치 (빨간 네모)]

### STEP 4. 저장
저장하면 자동으로 연결 확인합니다.
> ✅ 성공하면 📋 일일 주문 수집 탭에서 **🛒 쿠팡 주문 조회** 사용 가능

# 5️⃣ 기타 설정 (필요한 것만)

### 🔓 엑셀 비밀번호
네이버에서 받은 주문 엑셀에는 비밀번호가 걸려 있어요.
1. 스마트스토어에서 주문 엑셀 다운로드 → 파일 열 때 나오는 **비밀번호** 확인
2. 설정 탭 > 엑셀 비밀번호에 입력 후 저장 → 이후엔 자동으로 풀립니다.

> 💬 이 비밀번호는 네이버 **로그인 비밀번호와 다릅니다.** (엑셀 전용 비번)

### 📦 고정 비용 (택배비·박스비)
- **택배비**: 실제 내는 택배비 (기본 1,800원)
- **박스비**: 포장 박스값 (기본 300원)
- 수익 계산할 때 자동으로 빼줍니다.

### 📱 텔레그램 (카톡 대신 쓰고 싶을 때 · 선택)
1. 텔레그램에서 **@BotFather** 검색 → `/newbot` → **봇 토큰** 복사
2. **@userinfobot** 에 아무 메시지 → **Chat ID** 확인
3. 설정 탭 > 텔레그램에 입력 후 저장

# ❓ 막혔을 때 (자주 묻는 질문)

**Q. 저는 개발 업체(솔루션사)가 아닌데 API를 써도 되나요?**
> 네, 됩니다. 네이버 커머스 API는 **일반 판매자가 본인 계정으로 '내 스토어 애플리케이션'을 직접 발급**받아 쓸 수 있습니다(무료). 카카오·네이버 Open API·쿠팡도 개인/판매자 계정이면 충분하며, 별도의 개발 업체 계정이나 사업자 개발자 등록이 필요 없습니다.

**Q. 카카오 토큰 발급 때 'redirect_uri mismatch' 오류**
> 카카오 로그인 설정의 Redirect URI에 `http://localhost`가 **정확히**(끝에 `/` 없이) 들어갔는지 확인하세요.

**Q. '이미 사용된 코드'라고 나와요**
> 인가 코드는 1회용입니다. 카카오 로그인 링크를 **다시 눌러 새 코드**를 받으세요.

**Q. 네이버 커머스 API에서 '401' 오류**
> Secret을 다시 복사하거나, 앱 등록 때 **스마트스토어 API 체크**를 했는지 확인하세요.

**Q. 순위 체크에서 '검색 결과 없음'**
> 키워드를 더 짧고 일반적인 단어로 바꿔 보세요.

**Q. 엑셀 올릴 때 '비밀번호가 틀렸습니다'**
> 네이버 로그인 비번이 아니라 **엑셀 전용 비밀번호**입니다. 설정 탭에서 다시 확인하세요.

**Q. CJ대한통운 자동 접수가 안 돼요**
> 설정 탭 > 택배사 설정에 CJ ID/PW/고객번호가 들어갔는지 확인. 고객번호는 CJ 기업 고객센터(1588-1255) 문의.
